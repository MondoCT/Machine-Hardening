# PDF to Text Converter

This project provides a simple GUI tool to convert PDF files into text files while preserving as much formatting as possible. It utilizes the PyMuPDF library for reading PDF files.

## Features

- Extracts text from PDF files.
- Preserves basic formatting, including ASCII characters.
- Easy to use GUI interface.

## Requirements

- Python 3.6 or higher
- PyMuPDF

## Setup and Run

1. Download the `setup_and_run.py` script.

2. Run the script using Python:
   ```sh
   python setup_and_run.py
   ```

The script will clone the repository, install the required packages, and run the main application with the GUI. Follow the prompts in the GUI to select the PDF file and the output location for the text file.
